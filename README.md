<h1>Recursive Programming</h1>

<h2>Description</h2>
This program demonstraits the concept of recursive programming. The method "calcTotalCapacity" recursively calls itself as many times as necessary to calculate the total capacity.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Java</b> 
- <b>Eclipse IDE</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
